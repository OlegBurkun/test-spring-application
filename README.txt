# mySimpleSpringApp
Test application on spring boot and jsp


Auhtor Oleg Burkun.
My test web application.
Used technologies: Spring-boot, Hibernate, JSP, JS&Jquery, css, JUnit, MySql, etc.
Easy to scale and addition any new technologies.


To run this project:
1. checkout project from github
2. change mySql connection properties in file application.properties
3. run sql script ../resourses/db/db_dump.sql
4. run project with spring-boot
    in eclipse - run configuration --> spring boot app --> [check project, main class] --> apply --> run
    in command line - cd [youfolder] --> mvn spring-boot:run
5. try it on localhost:8080/
